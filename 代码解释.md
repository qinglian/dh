## 1. 项目概览

清炼导航是一个基于 React 的浏览器导航门户应用，提供两个核心视图：

| 视图 | 路由 | 功能 |
|------|------|------|
| **起始页 (start)** | `currentView = 'start'` | 搜索引擎选择 + 搜索框 + 快捷入口 + 多页面侧栏 + 时间/日期/问候语 |
| **导航页 (nav)** | `currentView = 'nav'` | 分类站点卡片 + 标签筛选 + 快捷访问面板 + 时间天气小组件 + 编辑模式 |

### 功能特性一览

| 功能 | 说明 |
|------|------|
| **动效背景** | 12 种 2D Canvas 动效 + 5 种 3D Three.js 动效 + 实时天气动画 |
| **壁纸背景** | 10 个内置渐变预设 + 自定义图片/GIF/视频上传 + URL 链接 |
| **磨砂玻璃** | 全局/独立窗口控制，可调模糊度、不透明度、文字颜色 |
| **鼠标聚光灯** | 跟随鼠标的径向渐变光源，支持双色混合、遮罩模式 |
| **编辑模式** | 拖拽排序（分类/站点/标签/搜索引擎）、撤销/重做（最多 50 步） |
| **多页面** | 起始页多页面系统，每页独立数据隔离 |
| **数据管理** | 导出/导入 JSON、WebDAV 云备份/恢复、选择性重置 |
| **保险箱** | 密码保护的隐藏站点列表，支持时间码找回 |
| **桌面宠物** | AI 行为引擎驱动的多宠物系统（猫/狗/鼠/鸟） |
| **天气系统** | 实时天气 API + Canvas 天气动画 + 日出日落时间主题切换 |

## 6. 上下文模块

### 6.1 `DataContext.jsx` — 导航数据中心

**导出：** `DataProvider`、`useData`

**默认数据：** 5 个分类 32 个预设站点（AI 工具 / 开发工具 / 设计资源 / 效率工具 / 影音娱乐）

**数据加载策略：**
1. 尝试从 `localStorage['nav-data-v2']` 读取 JSON
2. 校验 `categories` 是否为合法数组
3. 校验失败 → 回退到 `DEFAULT_DATA`
4. `useEffect` 中持续监控数据有效性

**CRUD 操作（全部通过 `useCallback` 记忆化）：**
| 方法 | 功能 |
|------|------|
| `addCategory(name)` | 新增分类，自动分配 `cat_` 前缀 ID |
| `updateCategory(id, updates)` | 更新分类名称 |
| `deleteCategory(id)` | 删除分类 |
| `addSite(catId, site)` | 新增站点，自动分配时间戳 ID |
| `updateSite(catId, siteId, updates)` | 更新站点信息 |
| `deleteSite(catId, siteId)` | 删除站点 |
| `resetData()` | 恢复为默认数据 |
| `reorderSites(catId, newOrder)` | 站点拖拽排序 |
| `reorderCategories(newCategories)` | 分类拖拽排序 |
| `reorderTags(catId, newTags)` | 标签拖拽排序 |

### 6.2 `ThemeContext.jsx` — 主题系统

**四种模式：**
| 模式 | 行为 |
|------|------|
| `light` | 始终亮色 |
| `dark` | 始终暗色 |
| `system` | 跟随系统 `prefers-color-scheme` 媒体查询 |
| `sunrise-sunset` | 根据日出日落自动切换 |

**主题切换优先级链（sunrise-sunset 模式）：**
```
天气 API 缓存数据 > 用户自定义时间 > 默认 06:00 / 18:00
```

设置 `document.documentElement.setAttribute('data-theme', ...)` 触发 CSS 变量切换，每 60 秒检查一次是否需要切换。

## 7. 工具模块

### 7.1 `weather.js` — 天气服务

使用 Open-Meteo 免费天气 API，无需 API Key。

**核心功能：**
- **WMO 天气编码表**（0-99）→ 中文描述 + 表情符号（☀️⛅🌧❄️🌫⛈）
- **获取天气**：`fetchWeather(lat, lon)` — 调取 Open-Meteo forecast API，30 分钟缓存
- **城市搜索**：`searchCity(keyword)` — Open-Meteo Geocoding API
- **浏览器定位**：`getLocation()` — 10 秒超时
- **反向地理编码**：`reverseGeocode(lat, lon)` — 从 64 个中国主要城市列表中查找最近城市

### 7.2 `siteStatus.js` — 站点连通性检测

**双通道检测机制：**

| 通道 | 方法 | 超时 |
|------|------|------|
| 图片通道 | 通过 Google S2 服务加载 favicon | 8 秒 |
| 请求通道 | `fetch(url, {mode:'no-cors'})` HEAD 请求 | 8 秒 |

**判定结果：**
- `online: true` — 任一通道成功
- `online: false` — 两通道均失败
- `unknown: true` — 两通道均失败 + URL 匹配已知需 VPN 访问的站点列表

**VPN/受限站点列表：** 30+ 域名（github.com, youtube.com, google.com, twitter.com, openai.com 等）

**检查策略：** 5 个并发一组的批量检查，30 分钟缓存

### 7.3 `webdav.js` — WebDAV 云备份客户端

**存储路径：** `/nav-portal-backup/nav-data.json`

**操作：**

| 方法 | HTTP | 说明 |
|------|------|------|
| `testConnection()` | `PROPFIND /` | 验证连接 |
| `backup(data)` | `PUT /nav-portal-backup/nav-data.json` | 备份数据 |
| `restore()` | `GET /nav-portal-backup/nav-data.json` | 恢复数据 |
| `getBackupInfo()` | `PROPFIND` | 获取备份文件信息 |

**认证：** HTTP Basic Auth（Base64 编码用户名:密码）

**预设服务：**
- 坚果云 (JiangGuoYun) — `https://dav.jianguoyun.com/dav/`
- 自定义 WebDAV — 用户自行填写

### 7.4 `quickAccess.js` — 快捷访问数据

四个子区域：

| 区域 | 排序逻辑 | 存储 |
|------|----------|------|
| 最近访问 | 按 `lastClick` 时间降序 | localStorage |
| 最常使用 | 按点击次数降序 | localStorage |
| 最新添加 | 按站点 ID (时间戳) 降序 | 遍历分类数据 |
| 已固定 | 用户手动固定 | localStorage |

`recordSiteClick(site)` 在每次点击站点时被调用，记录点击时间、次数和站点信息。

### 7.5 `safeBox.js` — 保险箱模块

密码保护的隐藏站点列表，支持时间码找回功能：

| 函数 | 说明 |
|------|------|
| `getSafeBoxEnabled()` / `saveSafeBoxEnabled()` | 开关状态 |
| `getSafeBoxPassword()` / `saveSafeBoxPassword()` | 密码存取（明文） |
| `verifySafeBoxPassword(input)` | 字符串相等校验 |
| `addSafeBoxSite(site)` | URL 去重加入 |
| `removeSafeBoxSite(id)` | 按 ID 移除 |
| `resetSafeBox()` | 清除所有保险箱数据 |
| `timeCodeMatch(code)` | 时间码校验（格式: HHMM） |

## 8. 核心组件详解

**状态管理结构：**

```

│   ├── animatedBg， bgEffects， bgMultiMode → 动效背景
│   ├── wallpaper， bgBlur， bgOpacity         → 壁纸
│   ├── mouseSpotlight， spotlightSize ...     → 聚光灯
│   └── blurEnabled， blurValue， opacityValue  → 玻璃效果
│
├── 多页面
│   ├── pages， currentPageId， pageData
│   └── 每页独立：{ categories， searchEngines }
│
├── 编辑模式
│   ├── isEditMode， preEditSnapshot
│   ├── historyStack（最多 50 步）、historyIndex

| gradient3 | 日冕 | 红橙黄紫多段暖色 |
| gradient4 | 森林 | 翠绿四段渐变 |
| gradient5 | 幻夜 | 深紫星空 |
| gradient6 | 晨曦 | 暖粉米色晨光 |
| gradient7 | 海雾 | 清冷蓝灰 |
| gradient8 | 梦境 | 粉蓝绿多彩 |
| gradient9 | 夜幕 | 深邃深蓝 |

**自定义背景存储格式：**

```

### 8.3 `TimeWidget.jsx` — 时间天气小组件

**Canvas 天气动画层（`WeatherEffect`）：**
- 雨：300 雨滴 + 飞溅粒子 + 涟漪 + 闪电
- 雪：180 雪花（六边形/圆形，分层远近）+ 地面积雪
- 雾：55 个雾块缓慢漂移
- 晴：太阳光晕 + 光线 + 云朵 + 飞鸟 + 尘土 + 夜间星空/极光
- 多云：20 朵云带 3D 阴影 + 高光
- 阴：28 朵慢速深色云

**天气数据源：**
1. `localStorage['nav-weather-cache']` — 30 分钟缓存
2. 缓存失效时自动调用 Open-Meteo API
3. 无法获取时使用内置模拟数据

### 8.7 `PetSystem.jsx` — AI 桌面宠物

**`PetAI` 行为引擎：**

```
属性: mood(情绪) energy(精力) hunger(饥饿) boredom(无聊) [0–100]
     
状态机:
idle → [随机] → chasing(追逐鼠标) / sleepy / hungry / playful / happy / love
                ↑                                                       │
                └──────────────── 属性变化触发 ─────────────────────────┘

宠物互动：
- 猫追鼠，鼠逃跑; 狗追猫; 鸟独立行为
- 拖拽时表情变化; 点击触发 happy/love/playful
- 投食降低饥饿值; 每帧属性自然衰减
```

### 8.8 `MouseSpotlight.jsx` — 鼠标聚光灯

**两种模式：**
- **非遮罩模式：** 全屏径向渐变圆跟随鼠标（0.12 lerp 缓动）
- **遮罩模式：** 圆裁剪到特定 DOM 元素（`[data-spotlight-group]`， `[data-spotlight-btn]`），精确匹配圆角

支持两个颜色之间的渐变混合（滑块 0-100%），可调大小和透明度。
